remember

A membrane-based Plone member implementation.

For questions and support, please see the remember mailing list:
http://www.openplans.org/projects/remember/lists/remember/

============
REQUIREMENTS
============

- Zope 2.9.3 (or greater)
- Plone 2.5.x
- membrane  (http://svn.plone.org/svn/collective/membrane/trunk)

Optional:

- contentmigration (if migrating an existing CMFMember site)
  (currently requires a special branch:
   http://svn.plone.org/svn/collective/contentmigration/branches/rossp-extensions)

- py-bcrypt (http://www.mindrot.org/py-bcrypt.html or 
             http://cheeseshop.python.org/pypi/bcrypt/0.1)
